# Build 115 Document Index

This is the recommended reading order for the Build 115 document set.

1. **`BUILD_115_ARCHITECTURE.md`** - Defines the canonical vocabulary and core
   invariants.
2. **`BUILD_115_SEAM_MAP.md`** - Defines ownership seams between analysis,
   banking, rendering, inspectors, presets, and UI shell work.
3. **`BUILD_115_WORKSPACE_SHELL.md`** - Defines the conceptual UI shell, panel
   taxonomy, and launcher behavior.
4. **`BUILD_115_FRAME_CONTRACTS.md`** - Defines `AnalysisFrame` and `BandFrame`.
5. **`BUILD_115_VISUALIZER_CONTRACT.md`** and
   **`BUILD_115_INSPECTOR_CONTRACT.md`** - Define the consumer interfaces for
   scene content and UI instrumentation.
6. **`BUILD_115_SCENE_MODEL.md`** - Defines `Scene`, `SceneNode`,
   `Compositor`, and `ViewTransform`.
7. **`BUILD_115_SCHEMA_PLAN.md`** - Defines schema implications and migration
   expectations for persisted scene configuration.
8. **`BUILD_115_PHASES.md`** - Defines the intended phase order for Build 115
   implementation work.
